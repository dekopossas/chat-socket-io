Criando uma aplicação react normal.

Colocando templete de https://codepen.io/sajadhsm/pen/odaBdd.

- Cors
- ejs
- express
- mongodb
- socket.io

### frontend

- axios
- react
- react-router-dom
- socket-io client
- web-vitals
